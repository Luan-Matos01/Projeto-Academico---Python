O que o projeto executa:
Um sistema completo de cadastro e gerenciamento de clínicas que permite:

Registrar e gerenciar dados de pacientes (criar, ler, atualizar, deletar)
Agendar e gerenciar consultas vinculadas aos pacientes
Buscar pacientes por nome
Buscar consultas por paciente ou especialidade
Listar todos os registros de forma ordenada
Gerar estatísticas sobre pacientes e consultas
Salvar e carregar dados em arquivo JSON
